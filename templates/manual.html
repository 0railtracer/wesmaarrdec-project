I. Introduction

A. purpose of the manual

This manual is designed to provide an overview of the admin system in our content management system. It will guide you through the process of managing user accounts, creating and editing content, and managing website settings.

B. Overview of the Content Management System

Our content management system is a web-based platform that allows you to create, manage, and publish digital content. It is designed to be user-friendly and customizable to meet your specific needs.

C. Admin System Overview

The admin system is the section of the content management system where you can manage user accounts, create and edit content, and manage website settings.

II. Getting Started

A. Accessing the Admin System

To access the admin system, navigate to the login page and enter your username and password.

B. Login Process

Once you have entered your username and password, you will be taken to the dashboard. The dashboard is the main hub of the admin system and provides an overview of your website's performance.

III. User Management

A. User Roles and Permissions

Our content management system includes three user roles: administrators, editors, and contributors. Each role has a different set of permissions and access levels.

B. Checking the total, active and inactive user

Our content management system can see the numbers of user/admin registered and can look if these specific user/admin is active or inactive

D. deleting user

To delete, navigate to the Users section of the admin system and click the "delete user button" button. also, if you are assigned as administrator you can delete it and it will indicate as Unavailable

IV. Content Management

A. Creating New Content

To create new content, navigate to the Posts or Pages section of the admin system and click the "Add New" button. You can then enter the content's title, text, and media.

B. Editing Existing Content

To edit existing content, navigate to the Posts or Pages section of the admin system and click the content's title. You can then make changes to the content's title, text, and media.

C. Deleting Content

To delete content, navigate to the Posts or Pages section of the admin system and click the content's title. Then, click the "Move to Trash" button and confirm the deletion.

D. Content Organizations

Our content management system allows you to organize your content into categories and tags. This makes it easier for users to find relevant content on your website.
